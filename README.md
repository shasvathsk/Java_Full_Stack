# Java_Full_Stack
This repository is a comprehensive workspace for learning and experimenting with Java Full Stack development. It contains structured folders for backend APIs, frontend projects, and openly available notes, making it both a practice ground and a reference hub.

Java_Full_Stack/
├── backend/          # Spring Boot practice APIs and examples
├── frontend/         # React, HTML, CSS practice projects
├── notes/            # Handwritten (digitalized) + typed notes
└── README.md         # Repository overview and usage guide

✨ Contents
Backend Folder

Multiple Spring Boot API examples for CRUD operations, authentication, and integration with PostgreSQL.

Practice with RESTful endpoints, entity mapping, and service layers.

Frontend Folder

React components, routing examples, and UI experiments.

HTML/CSS practice files for layouts, styling, and responsive design.

Notes (outside folders)

Digitalized handwritten notes and typed documentation.

Covers Java basics, Spring Boot concepts, React fundamentals, database integration, and full stack workflows.

🛠️ Tech Stack
Backend: Java, Spring Boot

Frontend: React, HTML5, CSS3

Database: PostgreSQL

Version Control: Git + GitHub

🎯 Purpose
This repository serves as:

A practice playground for backend and frontend development.

A knowledge base with notes for quick revision.

A portfolio showcase demonstrating both coding projects and study material.
